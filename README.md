## Como instalar?
link para crear app-->[Create React App](https://github.com/facebook/create-react-app).
Instalación para React Js:

 - npm install g create-react-app my-app

 - cd my-app

 - npm start

## Si ya usas React JS
 - npm install

 - La app corre como [http://localhost:3000](http://localhost:3000)
